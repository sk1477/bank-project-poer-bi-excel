# bank-project-power-bi-excel
 This project is about finding demography and churn out characteres of bank customers
The analysis highlights a clear contrast between customer groups. German females aged 45–60 with high product usage represent the most vulnerable segment, requiring targeted retention strategies. In contrast, younger French males with moderate product engagement remain stable and low‑risk. Understanding these patterns enables the bank to design more effective, data‑driven customer management initiatives.
